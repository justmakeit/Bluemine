
== What is SimpleWiki?

SimpleWiki is an easy-to-use input format based on the work done by WikiCreole
researching common wiki markup elements. SimpleWiki uses Creole syntax extended
with decorators and block declarations for advanced markup. For more 
information see:

* http://simplewiki.org/language
* http://wikicreole.org/
